# React Mobile Component

Current projects include component encapsulation for everyday use

## 目录结构

```js
components
├── BackTop                   # 返回顶部
├── Badge                     # 徽标 在右上角展示徽标数字或小红点。
├── Button                    # 按钮
├── Icon                      # 样式图标
├── Image                     # 图片组件
├── Loading                   # 加载组件
├── LoadMore                  # 加载完成 ｜ 加载中
├── Overlay                   # 遮罩层
├── Popup                     # 上下左右弹出
├── Switch                    # 开关
├── TextAlign                 # 文字对齐
├── Cascader                  # 数据选择 三级选择 支持 支持 数据格式化
├── Cell                      # 单元格为列表中的单个展示项。
├── ..............
```
